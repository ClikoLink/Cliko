Общая структура

HTML-документ index.html
<!DOCTYPE html> — объявляет стандарт HTML5, обязательно первой строкой
<html lang="ru"> — корневой элемент; lang указывает язык для поиска и скринридеров
служебная часть
<head> — браузер читает, пользователь не видит
/* НЕ видит пользователь, но нужно браузеру */
<meta charset="UTF-8"> — кодировка: поддержка кириллицы и всех языков
<meta name="viewport" content="width=device-width,initial-scale=1"> — корректное масштабирование на мобильных
<title>Заголовок</title> — текст на вкладке и в результатах поиска
<style>
/* цвета, шрифты, отступы — CSS внутри HTML */
body { margin: 0; }
</style>
CSS ?
Другие способы подключить CSS:

1. Внешний файл — самый правильный:
<link rel="stylesheet" href="style.css">

2. Атрибут style прямо на теге:
<p style="color: red">

3. <style> внутри <head> — как здесь. Удобен для небольших страниц.
</head> — конец служебной секции
видимая часть
<body> — всё внутри body видит пользователь
<header>  — шапка: логотип, навигация, баннер </header>
<main>  — основной контент страницы </main>
<footer>  — подвал: контакты, копирайт, ссылки </footer>
</body> — закрываем тело страницы
</html> — документ завершён
